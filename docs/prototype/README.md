# Protótipos — Laço App

Esta pasta foi criada para armazenar os protótipos de alta fidelidade e capturas de tela das versões finais do aplicativo.

> **Nota:** Como o projeto atual já está implementado e integrado com o Firebase e Google Maps, o link para o protótipo interativo no Figma está disponível no [README principal](../../README.md).

## Telas mapeadas

1. **Splash/Onboarding:** Animação de conexões suaves
2. **Login/Cadastro:** Fluxo minimalista com autenticação Firebase
3. **Home (Feed):** Linha do tempo estilo bento UI com os membros e momentos recentes
4. **Mapa:** Mapa limpo estilo Apple Maps com a localização dos membros e clima real
5. **Perfil:** Edição de dados, upload de foto e histórico pessoal de posts
6. **Criar Post:** Fluxo simples focado na foto, semelhante ao BeReal

> **Instruções para a entrega final:** Antes de zipar ou enviar o projeto, tire screenshots de cada uma destas telas rodando em um emulador iOS ou Android e salve-as nesta pasta.
